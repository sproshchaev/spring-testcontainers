Для версии 1.20.4 загружается testcontainers/ryuk:0.11.0
```

			<groupId>org.testcontainers</groupId>
			<artifactId>postgresql</artifactId>
			<version>1.20.4</version>  <!-- 1.19.8, 1.17.6 -->
			<scope>test</scope>
		</dependency>
```
